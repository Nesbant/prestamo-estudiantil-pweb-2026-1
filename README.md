# Préstamo Estudiantil - PWEB 2026-1

Aplicación de préstamos e intercambios entre estudiantes.

## Stack

- **Frontend**: React 19 + Vite + Tailwind CSS 4
- **Backend**: (Próximo)

## Setup

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Luego abre `http://localhost:5173`

### Build

```bash
cd frontend
npm run build
```

### Lint

```bash
cd frontend
npm run lint
```
